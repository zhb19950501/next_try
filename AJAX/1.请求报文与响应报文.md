# HTTP
HTTP(hypertext transport protocol) 协议 [超文本传输协议]，协议详细规定了浏览器和万维网服务器之间相互通信的规则约定

## 请求报文
重点是格式与参数

```
行  POST URL HTTP/1.1
头  HOST: atguigu.com
    Cookie: name=guigu
    .
    .
    名称： 值
空行
体  username=admin&password=admin
```
## 响应报文

```
行  HTTP/1.1 200 OK
头  Content-Type: text/html;charset=utf-8
    Content-length: 2048
    Content-encoding: gzip
空行
体  <html>
        <head>

        </head>
        <body>
            <p></p>
        </body>

    </html>
```
